# Learn Python

這個 Respository 並不單純只是為了我的好朋友學習 Python 而建立的，同時也是為了讓我自己複習和檢視過去在學習使用 Python 上是否有所遺漏，並且藉由這個機會瀏覽和搜集與 Python 相關的新資訊。

在這裡會儲存和分享整個過程中所有的 Python 範例，希望能對我自己，和對任何有需要的人有幫助。
